Abstract

SilentVoices is a real-time Indian Sign Language Recognition (ISLR) system that converts hand gestures into text and speech, enabling communication between the deaf-mute community and the general public. The system uses a webcam to capture hand gestures, processes them using MediaPipe hand-tracking to extract 21 hand landmarks, and classifies signs using a CNN model.

The recognized characters are assembled into words and sentences using debounce logic. The system also provides real-time word suggestions, multilingual translation, and text-to-speech output. The project was first developed as a desktop application and later extended into a full-stack web platform using Flask, Flask-SocketIO, and React.

1. Introduction

1.1 Background of the Project

Communication is one of the most essential aspects of human interaction. However, people with hearing and speech impairments often face barriers while communicating with those who do not understand sign language.

1.2 Problem Definition

Traditional communication between deaf-mute individuals and the general public is difficult because most people are unfamiliar with Indian Sign Language. Existing solutions are often expensive, hardware-dependent, or limited.

1.3 Motivation for the Project

The motivation behind SilentVoices is to create an assistive technology solution that empowers deaf-mute users to communicate independently.

1.4 Objectives and Scope of the Project

The project aims to develop a real-time webcam-based system for recognizing Indian Sign Language alphabets, building sentences, providing translation, and speech output.

2. Literature Review

2.1 Research and Existing Solutions in the Domain

Sign language recognition has evolved from hardware gloves to computer vision and deep learning based systems. Modern systems use CNNs and hand landmarks for better real-time performance.

2.3 How the Project Differs from Existing Work

SilentVoices is a software-only webcam-based system focused on Indian Sign Language, with sentence building, translation, speech output, and both desktop and web-based deployment.

7. Result

SilentVoices was successfully implemented as both a desktop and web-based system. The model achieved approximately 93–95% group-level accuracy and supported real-time sentence formation, translation, and speech output.

8. Challenges Faced

8.1 Development Challenges Encountered

Major challenges included gesture ambiguity, real-time inference, debounce tuning, MediaPipe compatibility, and frontend-backend synchronization.

8.2 Solutions or Workarounds Used

These issues were addressed using hand landmarks, heuristic rules, ProcessPoolExecutor, proper debounce logic, and ACK-gated communication.

9. Conclusion and Future Scope

9.1 Summary of the Project Achievements

SilentVoices successfully demonstrates a software-only Indian Sign Language recognition system that converts gestures into text and speech using a webcam.

9.2 Potential Areas for Improvement or Future Features

Future improvements include dynamic sign recognition, two-hand support, cloud deployment, mobile app development, and offline TTS.
